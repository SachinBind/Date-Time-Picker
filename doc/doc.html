Date & Time picker

Date & Time picker je program, napsaný v jazyce JavaScript. Umožňuje snadný výběr časových údajů (čas a datum) v HTML formulářích pomocí kalendáře.

Funkce

  • Výběr data pomocí kalendáře
  • Výběr času
  • Široké možnosti formátování časových údajů
  • Možnost zakázání přímé editace formulářového pole
  • Vyznačení aktuálního dne v kalendáři
  • Vyznačení vybraného dne v kalendáři podle data ve formulářovém poli

Rozšíření

Jako rozšíření byla implementována možnost vkládat časové údaje. Dalším rozšířením je použití formátování časových údajů jako v jazyce PHP, což umožňuje velmi široké možnosti formátování dle libovolných požadavků.

Podpora prohlížečů

Program funguje zaručeně na všech moderních prohlížečích (Internet Explorer 8, Mozilla Firefox, Opera, Safari, Chrome...).

Návod k použití

Date & Time picker slouží jako skript, který přidává k textovému poli formuláře tlačítko, pomocí nějž lze zobrazit pop-up okno pro výběr data/času. Pro správnou funkčnost je potřeba vložit do hlavičky HTML dokumentu JS soubor dateTimePicker.js a CSS soubor dateTimePicker.css. Dále je potřeba nahrát do adresáře dokumentu obrázky picker-date.png a picker-time.png do složky img. V HTML dokumentu vytvoříme formulář s textovým polem input a definujeme jednoznačný parametr id, který se dále použije ve volání funkce. Potom na konec dokumentu vložíme samotné volání funkce, které má tvar: 

datePicker(id, format, editable);
timePicker(id, format, editable);

Funkce datePicker vytvoří výběr pro datum a timePicker vytvoří výběr pro čas. Parametr id je řetězec, reprezentující id parametr textového pole. Parametr format je formátovací řetězec a editable je booleovská proměnná, která značí, zda bude pole editovatelné. Na začátku souboru dateTimePicker.js jsou definovány konstanty, které lze upravovat (např. pojmenování měsíců, dnů v týdnu a překlady všech použitých slov). Program lze tedy snadno přeložit do jiného jazyka. Všechny prvky kalendáře mají své třídy, a proto lze snadno upravovat jejich vzhled v CSS souboru.

Formátovací řetězce

Formátovací řetězce používají stejný syntax jako jazyk PHP: 

y ... rok jako dvojčíslí (00)
Y ... rok jako čtyřčíslí (2000)
L ... 0 (nepřestupný rok) nebo 1 (přestupný rok)
z ... číslo dne v roce (001-365)
m ... číslo měsíce (01-12)
n ... číslo měsíce (bez případné úvodní nuly, 1-12)
M ... anglická zkratka názvu měsíce (Jul)
F ... anglický název měsíce (July)
d ... číslo dne v měsíci (01-31)
j ... číslo dne v měsíci (bez případné úvodní nuly, 1-31)
t ... počet dní v daném měsíci (28-31)
S ... anglická koncovka čísla dne v měsíci (st, nd, rd, th)
D ... anglická zkratka názvu dne v týdnu (Mon)
l ... anglický název dne v týdnu (Monday)
w ... číslo dne v týdnu (0-6 nebo 1-7 v závislosti na verzi PHP - neděle má číslo 0 nebo 7)

h ... hodiny (01-12)
g ... hodiny (bez případné úvodní nuly, 1-12)
H ... hodiny (01-23)
G ... hodiny (bez případné úvodní nuly, 1-23)
a ... am/pm (dopoledne/odpoledne)
A ... AM/PM (dopoledne/odpoledne)
i ... minuty (00-59)
s ... sekundy (00-59)
U ... počet sekund od 1. ledna 1970
